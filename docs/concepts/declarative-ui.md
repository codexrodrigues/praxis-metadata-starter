# Declarative UI

## Definicao curta

Descrever o que renderizar e como se comportar em termos de metadados e
configuracao, deixando a infraestrutura decidir a renderizacao.

## Onde aparece no Praxis

- Frontend: `@praxisui/dynamic-form`
- Frontend: `@praxisui/table`
- Frontend: `@praxisui/core`
- Backend: `praxis-metadata-starter` com `@UISchema`

## Como aplicar

1. publique metadados ricos no backend
2. use componentes que interpretam schema no frontend
3. ajuste comportamento via configuracao e nao por duplicacao de componente

## Referencias publicas

- repositório público: `https://github.com/codexrodrigues/praxis-ui-angular`
- pacotes: `@praxisui/core`, `@praxisui/dynamic-form`, `@praxisui/table`
