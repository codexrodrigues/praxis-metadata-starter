# Examples

Esta pasta reune exemplos complementares para acelerar leitura humana e
construcao de prompts.

## Como usar junto com os guias

- primeiro siga a trilha principal em `docs/guides/`
- depois use os exemplos desta pasta apenas como reforco local

## Arquivos principais

- `EXEMPLO-PROMPT-CRUD-BULK.md`
- `filter-dto.md`
- `spring-integration.md`
- `stats-multi-metric-examples.md`

## Regra de uso para LLM

Os exemplos desta pasta nao devem substituir os guias principais nem exigir
consulta a app externo para fechar o fluxo.
