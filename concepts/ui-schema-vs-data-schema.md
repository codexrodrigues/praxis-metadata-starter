# UI Schema vs Data Schema

## Definicao curta

O data schema descreve a estrutura de dados. O UI schema descreve como esses
dados devem ser apresentados e editados.

## Onde aparece no Praxis

- Backend: `praxis-metadata-starter` publica ambos no contrato enriquecido
- Frontend: `@praxisui/core` normaliza o contrato
- Frontend: `@praxisui/table` e `@praxisui/dynamic-form` consomem os metadados
