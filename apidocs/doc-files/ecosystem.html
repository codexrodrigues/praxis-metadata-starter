Ecossistema - Projetos Relacionados

SDK Node, gerador x-ui, runtime Angular oficial e seed de backend para acelerar integracoes.

Node.js / TypeScript

Repositorio: codexrodrigues/praxis-node

SDK e ferramentas para integrar aplicacoes Node ao padrao de contratos e metadados do Praxis.

Gerador x-ui a partir de OpenAPI

Repositorio: praxis-openapi-ui-schema-generator

Converte documentos OpenAPI em artefatos JSON no vocabulario x-ui, uteis para UIs e pipelines.

Dica: aponte para seu /v3/api-docs ou para um arquivo .json/.yaml local.

Seed de Backend

Repositorio: praxis-backend-seed-app

Projeto inicial Spring Boot + Praxis com estrutura, configuracoes e exemplos para comecar corretamente.

Runtime Angular Oficial

Repositorio: praxis-ui-angular

Superficie publica das bibliotecas @praxisui/* para consumo de /schemas/filtered e do vocabulario x-ui.

Upload/Files Starter (Spring)

Repositorio: praxis-file-management

Starter de gestao de arquivos com validacoes de seguranca, limites, rate-limit, antivirus opcional e quotas.

<dependency>
  <groupId>br.com.praxis</groupId>
  <artifactId>praxis-files-starter</artifactId>
</dependency>

Endpoint de configuracao efetiva: GET /api/files/config.
