# Sandbox Recomendado

Use um projeto sandbox descartavel e isolado.

## Regras

- o sandbox deve nascer limpo
- a prova nao deve reaproveitar codigo pronto de outro app
- a LLM deve trabalhar apenas com os guias oficiais liberados para a rodada

## O que evitar

- rodar a prova dentro de um app ja existente
- reaproveitar classes, controllers ou DTOs prontos
- usar artefatos locais como "fonte de verdade" fora dos guias

## Objetivo

Garantir que o resultado reflita a qualidade dos guias e do contrato canonico,
nao a memoria de um projeto anterior.
